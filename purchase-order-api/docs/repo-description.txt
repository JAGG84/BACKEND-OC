
Purchase Order API · Node.js Starter Template
============================================

API REST profesional construida con Node.js, Express y MongoDB. Ideal para comenzar proyectos backend con autenticación JWT, control de roles, Swagger, pruebas y despliegue automático con GitHub Actions.

### ¿Qué incluye?

- 🔐 Autenticación con JWT y Refresh Tokens
- 👥 Control de acceso por roles (`admin`, `operator`, `viewer`)
- 📦 CRUD completo de órdenes de compra
- 📊 Reportes por estado, cliente y mes
- ✅ Validación con Joi
- 🧪 Testing automatizado con Jest y Supertest
- 🐳 Despliegue listo con Docker y docker-compose
- 🚀 CI/CD con GitHub Actions
- 🧰 Scripts de seed, colección Postman, y entorno staging
- 📝 Licencia MIT y guía de colaboración

Ideal como base para apps de logística, inventario, ventas o gestión empresarial.
